# 🐍 Basic Python Projects

A collection of beginner-friendly Python projects to practice core programming concepts like loops, conditionals, functions, and user input.

---

## 📂 Projects Included

### 🔢 1. Calculator (Simple)

* Performs basic operations: Addition, Subtraction, Multiplication, Division
* Handles division by zero errors

---

### 🎯 2. Number Guessing Game

* Generates a random number
* User guesses until correct
* Provides hints (too high / too low)

---

### 🔐 3. Password Generator

* Generates strong random passwords
* Customizable length and characters

---

### ✊✋✌️ 4. Rock Paper Scissors

* Play against the computer
* Randomized computer choices
* Displays winner after each round

---

### 📝 5. To-Do List (Simple)

* Add, view, and remove tasks
* Simple command-line interface

---

### 📊 6. Word Count Tool

* Counts words in a sentence or file
* Useful for basic text analysis

---

## 🛠️ Tech Used

* Python 3.x
* Standard Python Libraries

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/charanreddy0006/Basic-Python-Projects.git
```

2. Navigate to the folder:

```bash
cd Basic-Python-Projects
```

3. Run any project:

```bash
python filename.py
```
